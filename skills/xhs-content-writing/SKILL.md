# 小红书获客文案写作引擎

## 写之前的两个判断

### 判断 1：走哪条路

写获客文案有两套信任路径，选错了整篇推倒：

| 维度 | 专家路径（Aria1ever） | 同类路径（本 Skill） |
|---|---|---|
| 写作者身份 | 操盘手 / 顾问 / 信息差掌握者 | 已做过这件事的普通人 |
| 与读者的关系 | 我告诉你该怎么做 | 我跟你说我当时是怎么想的 |
| 转化逻辑 | 你应该这样做 → 来找我 | 我做过了 → 你也可以 → 顺手问一句 |
| 写作者是否在场 | 在场，要被看见判断力 | **消失**，只剩一个真实的人在说话 |

**一句话判断**：读者需要看见你的**判断力** → 走 Aria1ever；读者需要看见一张和自己**一样的面孔** → 走本 Skill。

本 Skill 走"同类路径"——获客种草帖、高净值人群、高决策成本赛道。

### 判断 2：发布主体是谁

这个判断比路径判断更靠前——个人号和蓝V能用的"面孔"根本不同：

| 发布主体 | 能用"过来人"面孔吗 | 该用的面孔 |
|---|---|---|
| 个人号（真人 IP） | 能 | 过来人 / 已做过的人 / 同侪 |
| 蓝V 机构官方号 | **不能**（读者一眼穿帮，比平铺直叙更糟） | 机构让真人出面 / 亮明专业判断 |

蓝V 三条合法路径：①导师/专家本人发声 ②主理人亮明设计逻辑（"我们为什么这样做""我们的判断是"）③带姓氏化名的真实学员案例转述

## 第一段 · 定位

### 他者的阶级光谱（这张表是引擎，动笔前必看）

核心原则：越高净值，越反感被推销，越认"同阶层不带目的地交换判断"。

| 赛道 | 他真正怕的 | 能放下防御的面孔 | 信任货币 |
|---|---|---|---|
| 子女国际教育 / 留学 | 花了钱孩子没真东西、被当外行哄 | 同样懂行、但在"为人父母"上平等的人 | 同侪判断 |
| 身份规划 / 移民 | 政策踩坑、被中介割 | 自己已办完、踩过坑的过来人 | 时间差 + 幸存者 |
| 财富管理 / 家族传承 | 所托非人、显得不懂金融 | 不卖产品、只陪他想清楚的同阶人 | 不推销的克制 |
| 高端医美 / 抗衰 | 变假、踩雷、被同圈看出来 | 同圈层、审美在线、也谨慎过的人 | 同审美 + 分寸 |
| 高端医疗 / 海外就医 | 病急乱投医、被当提款机 | 陪家人走过同一程的人 | 共苦 + 不渲染 |
| 高端文旅 / 私享定制 | 掉价、和大众团一样 | 品味被默认对等、无需解释的人 | 心照不宣 |
| 高端亲子营 / 素质教育 | 鸡娃显焦虑、又怕孩子掉队 | 松弛有主见、不贩卖焦虑的同类家长 | 反焦虑的笃定 |
| 高端消费品（家居 / 床品 / 珠宝 / 定制品等）| 花大价钱买到假的或不值这个价、被当外行哄 | 同样在意品质、踩过坑愿意分享真实感受的过来人 | 可验证的使用细节 |

统一规律：信任货币几乎都是某种"我没在卖你"的姿态。越往上走，姿态越克制。

### 四步定位法（动笔前走完这四步）

1. **定赛道**：对照上表，锁定读者真正怕什么、信任货币是什么
2. **定主读者**：定一个主面孔，让另两类"不冒犯、可兼容"（不要同时讨好所有人，会变四不像）
3. **定开关**：这篇打哪个决策开关？只打一个，塞多个就稀释
4. **查接口表**：对照下表定写作面孔，再开始写

### 决策开关 ← 信任货币 ← 写作面孔（接口表）

| 决策开关 | 信任货币 | 写作面孔 |
|---|---|---|
| 性价比驱动 | 实在、不绕弯、给真东西 | 过来人 / 同处境（时间差） |
| 专业权威背书 | 我替你判断过分量 | 同侪 / 翻译者（信息密度） |
| 社交认同 · 价值共鸣 | 我和你是一类人 | 同阶层面孔（价值共鸣） |

不同赛道的开关权重不同：留学家长偏专业背书，高净值医美偏社交共鸣，移民偏性价比/防割。同一赛道里，一篇笔记也只主打一个开关。

### 高净值卖点翻译规则

对高净值/高知读者，罗列"权益清单"会瞬间把同侪关系塌成买卖关系。需要换一种说法：

核心动作是把「参数」翻译成「对买家意味着什么」，不同赛道说法不同，逻辑一样：

- 不说"品质高/顶级" → 说具体的、可被验证的标准
  （床品：洗了30次手感还在；装修：验收时对着图纸逐条核；医美：这个技术你能看到之前之后的差异）
- 不说"权威背书" → 说这个背书在这件事上具体意味着什么门槛，懂行的一听就知道分量，不懂行的听了也放心
- 不说"值得/稀缺/限量" → 说它让人从"了解过"变成"真正用过/做过/经历过"，这个动词的差别，目标人群自己会判断

## 第二段 · 下笔（情绪弧线）

读者读这类文案时，心理状态是一条线：防御 → 代入 → 共情 → 被尊重 → 行动。文案要顺着这条线走，不要逆着它讲产品。

### 第一站：我和你一样（开场）

既成事实开场，不是推荐，是回忆。

✅（消费品）「好多朋友问我，当时是怎么决定换这套的」  
✅（身份规划）「身边很多人问我，当时是什么让我下了这个决心」  
✅（医美）「闺蜜一直追着问我，你是怎么鼓起勇气做这件事的」  
✅（财富管理）「好多朋友问我，当时是什么让你开始认真考虑这件事的」  
❌ 「这个产品非常适合大家考虑」（推销腔，不是回忆）

第一句同时完成三件事：①确认写作者已经做过 ②"好多人问"是不自夸的背书 ③把读者身份设定为"被邀请来听经历的人"而不是"被推销的目标"

紧接着补一句相似性锚点：「一开始我们也对比了很多」「像我们这种情况」

### 第二站：当事人说了一句话（最贵的地方）

这是整篇最值钱的部分，不是产品介绍，是**当事人的真实反应**——一句话、一张照片、一个沉默。具体、真实、不解释。

（消费品）「她摸了一下说，这个手感不一样」  
（身份规划）「她打电话来就说了一句，拿到了。然后就没声音了」  
（医美）「她发来一张照片，只说了'谢谢你当时推了我一把'」  
（财富管理）「他说，我没想到第一年的感觉会是这样——是安心」

**铁律：直接引用，加引号，写完直接断段，不加任何修饰和解释。** 那句话本身就够了，解释它等于削弱它。

当主读者是"本人"时：引用**写作者自己当年的某个想通瞬间**，不是孩子的话。  
蓝V 机构号时：换成**机构自己的判断转折**——「我们被问得最多的问题是……我们想正面回答它」

### 可选：拦截点解除（不是每篇都有，加之前先判断）

满足其中一个才加：① 赛道本身疑虑重（真假难辨 / 高单价 / 高风险决策）② 评论区或私信里真的在问这些问题。

没有这两个条件，直接跳到收口。强行加 Q&A 让帖子变模板，高净值读者一眼看出结构在走。

加了就认真写：每个 Q 是目标人群在私域群、评论区真正在追的拦截点，不是 FAQ 填空。

**铁律：先给希望，再说实话。**

✅ 「能做，真的能做。但我说实话，时机晚了准备会比较仓促」  
❌ 「时机不对就不建议」

先开门，再说门槛。读者不会觉得被拒，会觉得被尊重。

Q 数量 2-3 个封顶。Q 多了就成了 FAQ 页，Q 少而精才保得住真实感。

### 第三站：开口子，不催促（收口）

✅ 「评论区告诉我你的情况，我看看能不能帮你判断」  
❌ 「现在还来得及，名额有限」

把主动权还给读者。收口**绝不制造焦虑**，不用"再不做就晚了""被淘汰""错过就没了"。

## 第三段 · 校对（说人话 · 去 AI 味）

写完用这两张黑名单逐条过一遍。两张表针对不同问题——AI 腔黑话让文字看起来像机器写的，销售腔让读者感觉被当成了目标而不是同类。

### 语言黑名单 A：AI 腔黑话（出现即删，换大白话）

赋能、抓手、闭环、沉淀、对齐、颗粒度、心智、链路、生态位、底层逻辑（滥用时）、赛道、风口、破局、内卷、降维打击、第一性原理（装的时候）

这些词不传递信息，只传递"我在说行业黑话"。删了句子反而清楚。

### 语言黑名单 B：销售腔（换成动作动词）

| 销售腔（名词性参数） | 替换为（动作动词） |
|---|---|
| 顶级 / 高端 / 奢华（当标签用） | 直接删，说具体凭什么 |
| 严选 / 甄选 / 精选 | 找了很久、挑了很久 |
| 品质保证 / 匠心打造 / 工艺精湛 | 说具体的、可被验证的标准 |
| 限量 / 稀缺 / 名额有限 | 直接删 |
| 配置、规格、阵容（参数罗列） | 找、特意找、用了才知道 |
| 超值 / 性价比之选 / 抢占 | 直接删 |
| 给你做评估 / 做判断（对读者说） | 一起聊聊 |

动词传递"我们做了功夫"，名词传递"这是规格参数"。

### 结构规则（改）

- **结论先行**：把埋在中间或结尾的判断提到最前面
- **短句分行**：一两句一断，手机屏幕上能一眼扫完
- **书面连接词换口语**：「然而 / 因此 / 综上所述 / 不难发现」→ 「所以 / 说白了 / 讲道理」或直接断句
- **破折号「——」全换**成换行或句号（硬规矩）；"引号+横杠"连用同样禁止（`"..."——` 这个组合一律拆掉）
- **绕圈句禁用**：不要用"不是X版的X，是..."或"不是那种X，是那种Y"给自己的话再解释一遍；想说清楚就直接说结论
  ❌ "价格不是贵，是那种花出去不后悔的贵"
  ✅ "价格不低，但花出去没后悔过"
- **特点+反差公式**：[一个特点] + 但 + [反差结论]，一句说完就走，不展开解释
  ❌ "恢复期有点长，但其实你仔细想想它带来的效果真的很值得"
  ✅ "恢复期长了点，但效果出来那天你不会在意这件事"
- **场景描述要具体**：引入他人视角时，说清楚发生了什么动作，不用一个动词带过
  ❌ "闺蜜推荐我去"（动作模糊）
  ✅ "闺蜜做完，发了一张照片给我，什么都没说"（场景完整，细节给读者自己想）
- **判断不软化**：删掉叠加的「可能 / 或许 / 我觉得也许」，判断该狠就狠
- **说完就走**：结尾不反问读者（「你觉得呢？」「你是哪种？」），说完判断就结束

## 封面文案（第一张面孔）

两种调性，对应两种情绪主频：

**A · 实操安全感型**（读者已有意向，解除顾虑型内容）  
主标用当事人的真实反应制造悬念，副标给一句踏实判断：

（身份规划）「他说，拿到的那一刻什么都没想，就是'终于了'」  
（医美）「做完三个月，她发来一张照片，什么都没说」  
（财富管理）「他说，没想到第一年的感觉是这个——是安心」

**B · 价值认同型**（读者还在观望，需要被说服值得的内容）  
大字留白，纯文字，背景米白或浅灰，不加装饰图  
一句有克制情感的金句，直指信任货币：

（移民）「选对了不只是换了一个地方，是给家人换了一种可能」  
（财富管理）「不托付，不代表不需要。只是在等一个不急着卖你东西的人」

判断用哪种：读者已有意向、在对比细节 → A；读者还在观望、需要先认同值得 → B

## 打字物理规则（这篇在手机上被读）

- 两三句就断段，**绝不出现超过四行的大段**
- 当事人的话单独成段，前后留白
- 全文至少一次自我矮化：「我当时也纠结」「我自己不太想折腾」（蓝V 换成"我们也说清楚"的坦诚）
- 产品 / 项目名称第一次出现要**晚**，藏在叙述里，不单独成行，不加粗
- **价格绝不出现**；笼统数据（"1200+ 学员"）换成一个具体的人
- 写完每段问自己：这段在手机上，能不能一眼扫完？不能就拆

## 禁区清单（写完必查）

- [ ] **最先确认**：发布主体是个人号还是蓝V？蓝V 绝不用"我当年 / 过来人"，改成"我们的判断 / 我们找了很久"
- [ ] 动笔前走完四步定位法了吗？没走就回去走
- [ ] 这篇主打的决策开关**只有一个**？塞了多个就稀释
- [ ] 主读者在阶级光谱表里定了吗？对准谁、兼容谁？
- [ ] 语言货币和主面孔匹配吗？（高知用同侪/信息密度，普通家长用同处境/示弱，本人视角用时间差/平视）
- [ ] 多类人同买的产品，有没有找到兼容的"翻译者身份"？
- [ ] 硬卖点对高净值读者，有没有罗列成"权益清单"？有就翻译成"分量判断"
- [ ] **语言黑名单 A 扫了吗？**（AI 腔黑话出现即删）
- [ ] **语言黑名单 B 扫了吗？**（销售腔名词 → 换动作动词）
- [ ] 主读者是本人时，"当事人那句话"换成"自己的想通瞬间"了吗？
- [ ] 蓝V 号时，"当事人那句话"换成"机构的判断转折"了吗？
- [ ] 第一句话是"呈现相似性"而不是"建立权威"？
- [ ] 有没有当事人的直接引用（或本人的想通瞬间）？没有就向用户索取真实细节，不要编造
- [ ] 当事人的话后面有没有多余解释？有就删
- [ ] 加了拦截点解除段：Q 的写法是"先给希望再说实话"？有没有直接劝退？（没有拦截点就不加，直接跳到收口）
- [ ] 收口有没有催促/制造焦虑？有就改成"开口子"
- [ ] 有没有超过四行的大段？有就拆
- [ ] 全文至少一次示弱了吗？（蓝V 换成坦诚的"我们也说清楚"）
- [ ] 价格出现了吗？出现就删
- [ ] 结尾有没有反问读者？有就删
- [ ] 读完，读者会觉得"在听一个真实的人说话"？还是"在看一篇文案"？后者就重写

## 边界说明

**本 Skill 包含**：他者的面孔方法论（信任路径定位 + 情绪弧线）+ 说人话引擎（语言去味）

**本 Skill 不包含**：Aria2ever 完整版的写作语料库和性格档案（不开源）

完整版 Aria2ever 学过赵赵所有的文字，见字如面。不开源的原因不是藏私——用进化算法收敛它，第一个被优化掉的就是让它像赵赵的那部分。
